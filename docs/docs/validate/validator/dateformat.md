上面的数据是测试的数据提供者。


``` php
public function testBaseUse($value, string $format): void
{
    $validate = new Validator(
        [
            'name' => $value,
        ],
        [
            'name'     => 'date_format:'.$format,
        ]
    );

    $this->assertTrue($validate->success());
}
```
