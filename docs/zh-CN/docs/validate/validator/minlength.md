以下是通过的校验数据示例。


上面的数据是测试的数据提供者。


``` php
public function testBaseUse($value, $param): void
{
    $validate = new Validator(
        [
            'name' => $value,
        ],
        [
            'name'     => 'min_length:'.$param,
        ]
    );

    $this->assertTrue($validate->success());
}
```
