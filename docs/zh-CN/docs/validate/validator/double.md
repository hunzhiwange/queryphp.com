## 验证通过的数据

以下是通过的校验数据示例。

``` php
# Tests\Validate\Validator\DoubleTest::baseUseProvider
public function baseUseProvider(): array
{
    return [
        ['0.1'],
        ['-90.55'],
        ['1.0'],
        ['42'],
        ['1820'],
        [-42],
        [42],
    ];
}
```

上面的数据是测试的数据提供者。


``` php
public function testBaseUse($value): void
{
    $validate = new Validator(
        [
            'name' => $value,
        ],
        [
            'name'     => 'double',
        ]
    );

    $this->assertTrue($validate->success());
}
```
